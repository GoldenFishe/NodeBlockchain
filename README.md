# NodeBlockchain
Blockchain base on Node js.
### Version
    Current version: 'Development'
## INSTALLITION
```bash
    git clone https://github.com/GoldenFishe/NodeBlockchain
```
## START
```bash
    $ npm run start // start local server
```
## API
### Get all blocks in the blockchain
```
    GET '/blocks
```
### Mine new block
```
    POST '/mine', {data: 'your data'}
```
